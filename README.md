# English-manager - программа для изучения английского. При запуске программа потребует ввести ваше имя и выбрать сложность. В зависимости от выбранного уровня будет повышаться сложность слов. После прохождения теста у вас будет возможность увидеть свои результаты и если вы ответили хорошо, то сложность последующих слов будет повышена. В программе есть небольшое колличество достижений которые будут отображаться и сохраняться у вас в профиле.
